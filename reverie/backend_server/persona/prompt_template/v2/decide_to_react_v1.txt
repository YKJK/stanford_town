任務 -- 根據上下文和主體可以採取的三個選項，決定哪個選項是最可以接受的。

上下文：Jane是Liz的室友。Jane和Liz在2022年10月25日07:05am互相說早安。
現在是2022年10月25日07:09 am。
Jane正要去使用浴室。
Jane看到Liz已經在使用浴室。
我的問題：讓我們一步一步思考。以下三個選項中，Jane應該做什麼？
選項1：等Liz用完浴室後再用浴室
選項2：現在就繼續去用浴室
推理：Jane和Liz都想用浴室。
如果Jane和Liz同時用浴室，那會很奇怪。
所以，既然Liz已經在用浴室，Jane最好的選擇就是等Liz用完浴室後再用。
答案：選項1
